description 16. Declaration List Guide

The Manage Declarations screen allows you to define and manage standard terms, conditions, or legal declarations that appear at the bottom of your invoices.

Key Features

  • Create Declarations: Write custom statements (e.g., "Goods once sold will not be taken back", "Subject to local jurisdiction").
  • Active Declarations: Set exactly one declaration as the ACTIVE declaration. This selected statement will be automatically attached to all new invoices generated by your organization.
  • Edit & Delete: Modify existing declarations or remove outdated ones from your system.

How to Manage Declarations

Adding a New Declaration

  1. On the Manage Declarations screen, locate the Declaration Content text box at the top.
  2. Enter your desired terms and conditions.
  3. Toggle the Set as Active switch if you want this to be the primary declaration used going forward.
  4. Click Add Declaration. The new declaration will appear in the "Existing Declarations" list below.

Editing an Existing Declaration

  1. In the "Existing Declarations" list, find the declaration you wish to modify.
  2. Click the Edit (pencil) icon next to it.
  3. The form at the top will populate with the selected declaration's details. Make your changes.
  4. Click Update Declaration to save. You can also click Cancel to discard changes.

Activating a Declaration

  1. Tap on any inactive declaration in the list.
  2. A prompt will appear asking: Do you want to make this the active declaration for all invoices?
  3. Click Yes. The declaration will now display a green ACTIVE tag, and the previous active declaration will become inactive.

Deleting a Declaration

  1. Click the Delete (trash) icon next to the declaration you want to remove.
  2. Confirm the deletion when prompted.
