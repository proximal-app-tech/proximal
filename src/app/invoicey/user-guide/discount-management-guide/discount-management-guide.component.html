sell Discount Management Guide

The Discount Management screen allows you to create and manage promotional codes and coupons. These discounts can be applied to quotations and invoices during checkout to incentivize sales or offer special pricing to customers.

Here is a detailed breakdown of each field on the Discount Management screen:

1. Discount Name

Required

An internal label for your discount campaign.

  • Purpose: Helps you easily identify the discount from your list of promotions.
  • Usage: Enter a clear, descriptive name (e.g., Summer Sale 2026, VIP Customer Discount, Black Friday Promo).

2. Code

Required

The actual promotional code that will be used or entered during billing.

  • Purpose: This is the identifier applied to invoices to trigger the discount.
  • Usage: Enter a short, memorable code (e.g., SUMMER20, VIP15, SAVE10). This code will automatically be capitalized by the system.

3. Discount %

Required

The percentage amount to be discounted.

  • Purpose: Determines how much of the invoice total will be deducted.
  • Usage: Enter a number between 0 and 100. Note: All discounts are calculated as a percentage of the pre-tax subtotal.

4. Max Usage

Optional

A limit on how many times this specific discount code can be used in total across all your customers.

  • Purpose: Prevents a discount from being overused (e.g., limited to the first 50 customers).
  • Usage: Enter a number (e.g., 50). If left blank, the discount code can be used an unlimited number of times.

5. Start Date

Required

The date from which the discount becomes valid.

  • Usage: Select the date the promotion begins. By default, this is set to the current day.

6. End Date

Optional

The expiration date for the discount.

  • Purpose: Perfect for flash sales or seasonal offers.
  • Usage: Select a date when the discount should stop working. If left blank, the discount will never expire automatically.

7. Discount Status

Required (Toggle)

A switch to quickly enable or disable the discount.

  • Purpose: Allows you to temporarily pause a promotion without having to permanently delete it.
  • Usage: Toggle to Active to allow the code to be used. Toggle to Inactive to prevent the code from being applied to any new invoices.
lightbulb
Checking Status: The discount list will automatically display the real-time status of your codes, labeling them as ACTIVE, EXPIRED (if past the end date), MAXED OUT (if the max usage has been reached), or INACTIVE (if you manually disabled it).
