broken_image 12. Damaged Items Guide

The Damaged Items screen helps administrators manage inventory returns specifically marked as "Damaged". This feature ensures transparency in product stock handling after a customer returns a defective or broken item.

Key Features

  • Damaged Item Tracking: View a comprehensive list of all items returned with the "Damaged" status.
  • Detailed Information: Each record displays the item name, returned quantity, associated invoice number, customer name, and the date of return.
  • Rectify & Restock Workflow: Easily add rectified items back into your active inventory.

How to Restock a Damaged Item

If a damaged item has been repaired or rectified and is ready to be sold again, you can restock it:

  1. Locate the item in the Damaged Returns list.
  2. Click the Rectified & Restock button on the bottom right of the item card.
  3. A confirmation dialog will appear, asking if you are sure you want to add the specified quantity back to the available inventory.
  4. Click Restock to confirm.
  5. The system will automatically increase the product's available stock and update the return reason to "Damaged (Restocked)", removing it from the pending damaged list.
NOTE: The system fetches the latest data for these items based on their associated invoice's creation date (newest first).
