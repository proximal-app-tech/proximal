design_services Service Master Guide

The Service Master is the central hub for managing all the professional services your organization provides. By setting up your services here, you can quickly add them to invoices with automatically calculated prices, taxes, and discounts, ensuring accuracy and saving you time during billing.

Why Use the Service Master?

  • Speed: Instead of typing out service details manually for every invoice, you simply select the service from a dropdown.
  • Compliance: By associating the correct SAC Code and Tax Rule with a service, you guarantee that your invoices remain GST-compliant.
  • Consistency: Ensures uniform pricing across your organization.
  • Reporting: By assigning services to specific Categories, you can generate organized revenue reports later.

How to Add or Edit a Service

To add a new service, click the + icon (on mobile) or the ADD NEW SERVICE button (on desktop). To edit an existing service, click the Edit (pencil) icon next to it in the list.

You will need to fill out the following fields in the Service Form:

1. Service Name (Required)

The name or title of the service you are offering (e.g., Website Development, Consultation Fee, Annual Maintenance). This is exactly how it will appear on the final invoice.

2. SAC Code

Services Accounting Code (SAC) is a classification system used by the GST department to identify services. Entering the correct SAC code is highly recommended for GST compliance on B2B invoices.

3. Service Price (Required)

The base price of the service before any taxes or discounts are applied.

4. Unit

How do you measure and bill this service? You can select from the following dropdown options:

  • Hour (HR)
  • Day (DAY)
  • Month (MONTH)
  • Project (PROJECT)
  • Per Unit (UNIT)

5. Category (Required)

Select the group this service belongs to. Grouping services helps keep your catalog organized and is essential for detailed sales reporting. (Note: Categories must be created in the Category Master before they can be assigned here).

6. Tax Rule (Required)

Assign the applicable GST rate (e.g., GST 18%) to the service. When you add this service to an invoice, the system will use this rule to calculate the tax automatically based on the customer's state. (Note: Tax rules must be set up in the Tax Configuration screen).

7. Auto Discount Code (Optional)

If you run specific promotions, you can attach an active discount code to this service.

  • None (Inherit from Category): The service will default to whatever discount is applied to its parent category.
  • Specific Discount: Overrides the category logic, applying a distinct percentage discount to this specific service whenever it is billed.

Managing Your Services

  • Search: Use the search bar at the top of the screen to quickly find a service by its Name or its SAC code.
  • Delete: Click the Delete (trash bin) icon to remove a service from your master list. Note: Deleting a service will not affect past invoices that have already been generated, but it will remove it from the selection list for future invoices.
